Practice App Help

Study and recite the Amarakosha with Listen, Recite, and Guided modes.

Overview

The Amarakosha Practice App lets you work through the pādas of the Nāmaliṅgānuśāsana at your own pace. You can listen to reference audio, recite from memory, study word-by-word, and build custom practice playlists called Practice Sets.

The text is organised as: Kāṇḍa → Varga → Section → Pāda. There are 3 Kāṇḍas, 25 Vargas, 63 Sections, and 2,865 Pādas in total. Each Varga is a topical group (e.g. heavenly bodies, trees, animals); each Section is a semantic cluster within a Varga.

Home Screen

The home screen shows a resume card for your current position, three quick stats (total pādas, sections, vargas), and a list of all Kāṇḍas with pada counts.

  • Tap Continue Practice or any Kāṇḍa row to go to the Browse screen.
  • The Browse screen lets you pick a Varga to enter the Practice screen.

Browsing the Text

The Practice screen has three areas:

  • Tree pane (left sidebar on desktop, slide-in overlay on mobile) — the full hierarchy of Kāṇḍas, Vargas, Sections, and Pādas.
  • Practice pane (centre) — the currently selected pāda, audio player, and word chips.
  • Practice Set panel (right sidebar on desktop, bottom sheet on mobile) — your curated playlist.

Tree pane navigation

  • Tap a Kāṇḍa row to expand or collapse it.
  • Tap a Varga row to expand it and see its Sections.
  • Tap a Section to jump to its first Pāda.
  • When a Section is selected, its individual Pādas appear below it — tap any to go directly to it.
  • A small speaker icon (🔊) next to a Pāda means audio is available for it.
  • A small dot (●) means Devanagari text is available.
Long-press a Section or Pāda in the tree to add it to your Practice Set. A checkmark icon (✓) appears on items already in the set.

Practice pane navigation

Use the ← → skip buttons in the audio player to move to the previous or next pāda within the current section. Navigation wraps across section boundaries.

The progress bar and "Pāda N of M" label at the top show your position within the current section.

Practice Modes

Three modes are available via the segmented control at the top of the practice pane:

Listen Recite Guided

Listen

The full Devanagari text is always visible. Use this mode to read along while audio plays, or to familiarise yourself with the pāda before memorising it. Audio plays on demand — tap the play button.

Recite

The Devanagari text is hidden initially, asking you to recite the pāda from memory. Tap Reveal (or the hidden text area) to uncover it and check your recitation. Use audio playback before or after revealing to hear the correct form.

Guided

The text is hidden but can be toggled on and off with a tap — useful for a prompt-then-reveal study method. Tap the shaded area to reveal; tap again to hide.

Audio Playback

The Audio card at the bottom of each pāda contains the full playback controls.

◀ ⏹ ▶ / ⏸ ▶ skip prev · stop · play/pause · skip next
0.5× 0.75× 1× 1.25× 1.5× playback speed
  • Play/Pause — starts or pauses audio for the current pāda.
  • Stop — stops playback and resets the progress bar to the beginning.
  • Skip buttons — navigate to the previous or next pāda.
  • Speed chips (0.5× – 1.5×) — adjust playback rate without re-loading.

Audio Sources

Below the progress bar is a Source row with chips for each available audio source:

Built-in Teacher A × + Import
  • Built-in — the reference recordings bundled with the app (where available).
  • Named sets — teacher recordings you have imported from a ZIP file. Tap a set chip to activate it; the × button removes the set from the app.
  • + Import — opens the import dialog (see Teacher Audio Sets).

The active source chip is highlighted. If the selected source has no recording for the current pāda, the app automatically falls through to the next available source.

Auto-advance

The Auto-advance toggle (below the speed chips) makes the app automatically move to the next pāda and begin playing as soon as the current pāda finishes. Useful for continuous listening without touching the screen.

Practice Set

A Practice Set is a curated playlist of Sections and/or individual Pādas that you want to drill. It persists across sessions (saved locally in the browser).

Adding items

  • Long-press a Section in the tree → adds all pādas in that section.
  • Long-press a Pāda in the tree → adds just that pāda.
  • Tap the + Set button on the pāda card → adds the current pāda.

Managing the set

On desktop, tap the Set (N) button in the nav bar to open the Practice Set panel on the right. On mobile, tap the set tray bar at the bottom of the screen.

  • Drag items to reorder them.
  • Tap × on an item to remove it.
  • Tap Clear to remove all items.

Repeat settings

  • Each Pāda — how many times each pāda is repeated before advancing.
  • Full Set — how many times the entire set is cycled through.
  • The total pāda count is shown below the steppers.

Playing the set

Tap Play Set to begin sequential playback through all items. The practice pane navigates automatically to the current pāda as the set plays. Tap Stop Set (red button) to end playback at any time.

Tip: Set playback uses the currently active Audio Source. Switch to a teacher set before tapping Play Set to drill with the teacher's voice.

Teacher Audio Sets

A teacher can record pādas in the Recording Studio and export them as a ZIP file. You can import that ZIP here as a named Audio Set.

Importing a ZIP

  1. In the Audio card, tap + Import.
  2. A dialog appears showing how many WAV recordings were found in the ZIP.
  3. Choose Create new set and enter a name (e.g. "Pandit Sharma"), or choose Update existing set to replace recordings in an already-imported set.
  4. Tap Import. The set appears immediately in the Source row.

Using a set

Tap the set's chip in the Source row to activate it. All playback (individual pādas and Practice Set) will now use the teacher's recordings where available, falling back to Built-in for any pādas not covered by the import.

Removing a set

Activate the set, then tap the × that appears on its chip. This permanently deletes the imported recordings from the browser's storage.

Audio sets are stored in the browser using IndexedDB. They persist across sessions but are specific to this browser and device. To use a set on another device, import the ZIP there as well.

Recording Your Own Voice

Tap the 🎙 Record button in the nav bar (or in the Audio card) to open the Recording screen for the current pāda.

The Recording screen uses your device's microphone to capture a recitation. Recordings are saved locally and appear as a source option in the Audio card. This is useful for self-assessment — record your recitation, then compare it to the reference.

For creating and distributing recordings to students, use the dedicated Recording Studio instead — it has export features and a full varga-level workflow.

Word Analysis

Where word analysis data is available, a Words section appears below the pāda card. Tap the "Words N" heading to expand it.

Each word chip shows:

  • The Devanagari form of the word
  • A brief meaning or grammatical label
  • The IAST transliteration (italicised)

Long-press a word chip to open a detailed grammar sheet with the full analysis, stem form, and grammatical category.

Display Settings

Tap the ⊞ (font/text) icon in the top-right of the nav bar to open Display Settings. You can adjust:

  • Devanagari font size — the size of Sanskrit text in the pāda card.
  • UI font size — the size of English labels and section titles.
  • Devanagari font — choose between bundled fonts for optimal rendering.
  • Context lines — show 0–3 neighbouring pādas (greyed out) above and below the current one for contextual reading.

Mobile Tips

On small screens the layout adapts:

  • The tree pane is hidden by default. Tap Contents in the nav bar to open a full-screen slide-in tree. Tap outside it (or the × button) to close.
  • The Practice Set panel becomes a bottom sheet. Tap the Practice Set tray bar at the bottom to open it.
  • Audio controls are slightly larger for easier tapping.
  • The Set toggle button in the nav bar is hidden — use the bottom tray instead.
Install as an app: On Chrome or Safari, use Add to Home Screen (or Install in Chrome's address bar) to run the Practice App as a standalone app without browser chrome. It works offline once cached.
